Function Checklist

1. did you start your function with 'def'?

2. does your function have only characters and _(underscore) characters?

3. did you put a parenthesis ( right after the function name?

4. did you put  your arguments after the parenthesis ( separated by commas?

5. did you make each arguent unique (meaning no duplicated names).

6. did you put a close parenthesis and a colon ): after the arguments?

7. did you indent all lines of code you want in the function 4 spaces? no more no less.

8. did you "end" your function by going back to writing with no indent (dedenting we call it)?

-----

Run (call/use) Function Checklist

1. did you call/use/run this function by typing it's name?

2. did you put ( character after the name to run it?

3. did you put the values you wanted into the parenthesis separated by commas?

4. did you end the function call with a ( character?
